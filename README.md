# OZE Warszawa

Lokalny portal informacyjny o odnawialnych źródłach energii w Warszawie i na Mazowszu.

## Stack
- Astro 4.x (static output)
- Vanilla CSS (dark navy + gold theme)
- Cloudflare Pages

## Uruchomienie lokalne
```bash
npm install
npm run dev
```

## Deploy
Cloudflare Pages: build command `npm run build`, publish directory `dist`
